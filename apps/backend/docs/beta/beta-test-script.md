# FISH_MARKET Beta Test Script

Please follow the script for your specific role. Report any issues using the in-app feedback widget.

## ⚓ Role: Boat Agent (3 Testers)
**Mission: Accurate high-speed data entry in the harbor.**

1. [ ] **Manual Entry**: Add 3 different fish catches (e.g., Sankara, Vaaval) manually.
2. [ ] **Voice Entry**: Try adding a catch using Tamil voice command ("Sankara 20kg 250rs").
3. [ ] **Offline Mode**: Turn off Data/Wi-Fi, add 2 catches, then turn it back on. Verify they sync!
4. [ ] **Receipt**: Generate a PDF slip and verify the ₹ total is correct.

## 🏢 Role: Market Owner (3 Testers)
**Mission: Financial oversight and reporting.**

1. [ ] **Dashboard Check**: Verify the 7-day sales trend looks accurate.
2. [ ] **Report Generation**: Download a monthly settlement report for a specific boat.
3. [ ] **User Management**: Verify you can see all slips sent by your agents today.
4. [ ] **Language Toggle**: Switch between Tamil and English. Verified all text translates correctly?

## 💰 Role: Buyer (2 Testers)
**Mission: Transparent purchasing and receipts.**

1. [ ] **Invoice View**: Open a shared invoice link on your phone.
2. [ ] **PDF Clarity**: Download the PDF and check if the font is readable and ₹ formatting is correct.
3. [ ] **Payment Tracking**: Verify that your recorded payments reflect correctly on the slip.

## ⚡ Role: Power User (2 Testers)
**Mission: Break the app.**

1. [ ] **Stress Test**: Open multiple tabs, add rapid entries.
2. [ ] **Conflict Test**: Try editing the same catch on two different devices.
3. [ ] **Edge Cases**: Enter zero weights, negative prices, or very long fish names.
